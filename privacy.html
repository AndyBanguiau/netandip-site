Politique de confidentialité

Dernière mise à jour : 16 juin 2026

1. Données collectées

NetAndIP ne collecte aucune donnée personnelle. L'application fonctionne entièrement en local sur votre machine Windows. Aucune information n'est transmise à des serveurs tiers dans le cadre de son utilisation normale.

2. Données stockées localement

Les fichiers suivants sont créés sur votre poste uniquement :

  • config.json — sites, rôles et adresses IP de votre organisation
  • profiles.json — configurations réseau sauvegardées
  • settings.json — préférences d'interface (mode Simple/Expert)
  • license.json — clé de licence Pro (si activée)

Ces fichiers restent exclusivement sur votre machine. Ils ne sont jamais lus, transmis ou analysés par l'éditeur.

3. Vérification des mises à jour

Si la vérification des mises à jour est activée (option activée par défaut, désactivable dans les paramètres), l'application contacte l'API publique GitHub Releases une fois par jour pour vérifier si une nouvelle version est disponible.

  • URL consultée : https://api.github.com/repos/AndyBanguiau/netandip/releases/latest
  • Seule la version courante de l'application est incluse dans le User-Agent HTTP standard.
  • Aucune donnée personnelle, aucun identifiant unique n'est inclus dans cette requête.
  • Cette fonctionnalité peut être désactivée dans Paramètres → Vérifier les mises à jour.

4. Cookies et traceurs

NetAndIP (l'application) n'utilise aucun cookie, aucun traceur, aucune balise de suivi. Ce site web statique n'utilise pas non plus de cookies analytiques ou publicitaires.

5. Droits des personnes (RGPD)

Conformément au Règlement (UE) 2016/679 (RGPD), toute personne dispose de droits sur ses données personnelles. Étant donné qu'aucune donnée personnelle n'est collectée par l'éditeur, il n'existe pas de traitement de données personnelles au sens du RGPD.

Vos données locales (fichiers .json) vous appartiennent intégralement. Vous pouvez les consulter, les modifier ou les supprimer à tout moment en accédant aux fichiers dans le dossier d'installation.

6. Sécurité de la licence Pro

Si vous activez une licence Pro, la clé de licence est stockée dans license.json sur votre machine. Elle contient votre adresse e-mail, le niveau de licence, la date d'expiration et une signature cryptographique. Ce fichier n'est jamais transmis automatiquement. La vérification de la licence est effectuée entièrement hors-ligne.

7. Contact

Pour toute question relative à cette politique ou à la protection de vos données : banguiauandy@gmail.com
