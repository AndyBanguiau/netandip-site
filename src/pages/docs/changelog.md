# Historique des versions

## v3.0.0 — Juin 2026

**Majeure — Rebranding NetAndIP + Système freemium**

### Nouveautés
- Nouveau nom : NetAndIP (anciennement NetPilot)
- Nouveau système de licences freemium (Gratuit / Pro)
- Génération de licence Ed25519 cryptée
- Validation de licence offline
- Nouvelle interface "Mode Expert" complète
- Import/Export amélioré des profils et configurations
- Vérification automatique des mises à jour intégrée

### Améliorations
- Interface redessinée avec thème sombre premium
- Navigation par vues avec barre latérale
- Gestion des cartes réseau filaire et Wi-Fi unifiée
- Détection automatique de l'interface active

### Corrections
- Correction du basculement DHCP/IP statique
- Meilleure gestion des erreurs netsh
- Stabilité améliorée sur Windows 11
